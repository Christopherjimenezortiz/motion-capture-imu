# 6-Axis IMU Sensor Node

Real-time motion sensing with ESP32-C3 and BMI160 sensor.

## Features
- Accelerometer (X, Y, Z) reading via SPI
- Gyroscope (X, Y, Z) reading via SPI
- 10Hz sampling rate
- ESP-IDF framework

## Hardware
- ESP32-C3 Super Mini
- BMI160 6-axis IMU breakout
- SPI communication (MOSI, MISO, CLK, CS)

## Status
Sensor reading functional  
BLE wireless transmission in progress  
PCB design planned

## Part of
6-node wireless motion capture system for AI-driven movement analysis.
